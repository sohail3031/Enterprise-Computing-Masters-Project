Introduction

In the current digital era, movie recommendation systems plays a key role in improving user experience and engagement. With the use of machine learning techniques like TF-IDF, KNN (K-Nearest Neighbors), and cosine similarity, this project attempts to create a Java-based movie recommendation engine. When a user searches for a movie, the system makes use of these algorithms to suggest films from the dataset that are similar to the search keyword. Inspired by the demand for personalized recommendations in the entertainment industry, this project serves as a practical exploration into data-driven solutions.

Problem solving and algorithms

The dataset, sourced from various movie databases, includes movie titles, genres, ratings, and other relevant information. Using Java programming language, we’ll implement cosine similarity, TF-IDF, and KNN algorithms to analyze textual data such as movie titles and genres. These algorithms will help recommend movies based on user search queries.

Proposed System Design

  1. Search Interface: Users can input a movie name into the search bar.
  2. Recommendation Display: Upon submitting a search query, the system will use cosine similarity, TF-IDF, and KNN algorithms to display movies from the dataset that are similar to the search keyword.
  3. User Interaction: Users can interact with the recommended movies, exploring additional details such as genre, rating, and synopsis.
  4. About Us Page: Provides information about the project and contributors.

Preliminary Technical Tools:

The project will be developed using Eclipse IDE for Java Developers and MySQL for database management. We’ll utilize Java libraries for implementing cosine similarity, TF-IDF, and KNN algorithms. Integration with Java EE for web development will ensure a seamless user interface.
